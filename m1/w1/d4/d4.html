COME CREARE UN TEMPLATE PER I POST

Di Mario Rossi Pubblicato il 25-10-2022
Come prima cosa che bisogna fare è suddividere la pagina in blocchi logici, identificando le parti comuni da applicare tutte le pagine.

    Le varie aree saranno suddivise in:

  • contenuti di intestazione del tag < Header >
  • contenuti centrali variabili dai tag < Main > (corpo principale)
  • contenuti di fine pagina da inserire nei tag < Footer >
Il testo dell'articolo dovrà essere formattato per identificare tutte le parti da selezionare con il CSS .
    Perciò i passi saranno:
  • Progettare su carta il markup finale
  • Fare altre robe
  • Applicare i stili CSS

    • L'applicazione di stili css è cresciuta negli anni fino a raggiungere la diffusione totale
